### 车场权限

- 从《车场权限.docx》复制出来的内容

1. 新建车场管理
2. 用户信息新增/修改，增加车场权限下拉框选择(可多选)
3. 超级管理员可查看/操作所有车场数据信息，其余用户根据用户与车场关联关系做权限判断进行相应车场信息查看/操作

#### 用户和车场

- 用户选择车场
- 新增用户车场关联表
- 

#### 用户绑定车场流程

1. 新增部门，车场管理勾选为是
2. 新增部门下用户
3. 新增车场，选择负责人，只能从勾选的部门下面选
4. 返回新增用户，修改车场

#### 逻辑

- 一个车场只有一个负责人
- 一个负责人可以有多个车场
