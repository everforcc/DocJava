### 7. 编写定时任务需要执行的备份脚本

~~~
# 保存到指定目录内 
/mnt/hkzk/crcc_shell/mysql/test/backup.sh
# 需要开启权限
chmod +x backup.sh
~~~

### 8. 设置定时任务

~~~
crontab -e
# 每分钟
*/1 * * * * /mnt/hkzk/crcc_shell/mysql/test/backup.sh  >> /var/log/cron.log 2>&1
# 每天 凌晨两点执行定时任务
0 2 * * * /mnt/hkzk/mysql/shell/backup.sh  >> /var/log/cron.log 2>&1
~~~
