### 浏览器前进后退

- 假设你依次访问了以下页面：

~~~
A -> B -> C
~~~

- 此时：
    - 后退栈内容为：[A, B, C]（C 是栈顶）
    - 前进栈为空
- 当你点击“后退”一次：
    - 跳转到页面 B
    - 后退栈变为：[A, B]
    - 前进栈变为：[C]
- 再点击“前进”：
    - 跳转回页面 C
    - 后退栈恢复为：[A, B, C]
    - 前进栈变为空
