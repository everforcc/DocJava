### C10K问题

- [服务器如何支持10k个并发连接](https://blog.csdn.net/chinawangfei/article/details/102780959)
- 也就是concurrent 10000 connection

### 解决方案-发展历史

- 每一个链接分配一个独立的线程/进程
- 同一个线程/进程同时处理多个连接（IO多路复用）
    - socket
    - select
    - poll
    - epoll
