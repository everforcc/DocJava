### maven

~~~
- maven编译
- -P,--activate-profiles <arg>           Comma-delimited list of profiles to activate
- -D,--define <arg>                      Define a system property
- maven.test.skip 跳过单元测试
mvn clean package -P bdy -D maven.test.skip=true
~~~
