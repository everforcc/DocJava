- BCrypt

### 最终完善版完整总结

1. 生成随机盐值（包含 BCrypt 版本标识、加密强度（cost factor）、16 字节随机数，格式如 $2a$10$abcdefghijklmnopqrstuv）；
2. 将盐值与用户明文密码结合，通过 BCrypt 算法进行多次哈希迭代运算（迭代次数为 2^ 强度值），生成唯一的哈希结果；
3. 将「版本 + 强度 + 盐值 + 哈希结果」按固定格式拼接成完整的复合字符串（如 $2a$10$abcdefghijklmnopqrstuv12345678901234567890123456789012），存储到数据库；
4. 用户登录输入密码后，从数据库取出该用户对应的复合字符串，按 $ 分隔并截取其中的盐值（版本 + 强度 + 随机盐部分）；
5. 用提取出的完整盐值（包含版本和强度），调用 BCrypt 算法对用户本次输入的明文密码执行与加密阶段完全相同的哈希运算，生成一个新的完整复合字符串；
6. 逐字符对比 “新生成的复合字符串” 和 “数据库存储的复合字符串” 是否完全一致，一致则密码验证通过，不一致则验证失败。
