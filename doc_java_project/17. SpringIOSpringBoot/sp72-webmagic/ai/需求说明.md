### 需求说明

- 需求描述：
- 小说阅读器，实现小说简介查询，书架列表查询，书架分页，章节列表查询，章节分页，章节内容详情页面

###  具体页面

- 使用jquery + bootstrap

#### 页面一

- 小说简介
- 需要包含，书籍名称，封面图，简介，目录
- 点击目录，跳转到小说章节列表页面

#### 页面二

- 小说章节列表
- 需要包含，章节名称，章节序号，章节内容
- 根据章节序号，从小到大排序
- 点击章节，跳转到小说章节内容页面

### 页面三

- 小说章节内容
- 需要包含，章节内容，上一章，下一章，如果是最后一章，则下一章为空，如果是第一章，则上一章为空
- 需要包含，目录，书架
- 点击目录，跳转到小说章节列表页面
- 点击书架，跳转到书架列表页面
