### 1. 表级数据操作权限

- SELECT：查询表中的数据
- INSERT：向表中插入数据
- UPDATE：更新表中的数据
- DELETE：删除表中的数据

### 2. 表级结构操作权限

- ALTER：修改表结构
- INDEX：在表上创建或删除索引
- DROP：删除表

### 3. 数据库级权限

- CREATE：在数据库中创建新对象（表、视图等）
- ALTER：修改数据库结构
- DROP：删除数据库对象

### 4. 特殊权限

- ALL PRIVILEGES：授予对象上的所有权限
- WITH GRANT OPTION：允许被授权者将权限再授予他人

### 5. 其他常见权限（虽然文档未明确提及）

- REFERENCES：创建外键约束
- EXECUTE：执行存储过程或函数
- USAGE：连接到数据库（无具体操作权限）
- 这些权限可以根据不同的数据库对象（表、视图、数据库等）进行灵活组合和授予，以实现精细化的权限控制。
