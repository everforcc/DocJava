- 位是最小的单位，字节是基本存储单位，字与 CPU 性能相关，而 KB 及以上单位则用于表示更大的存储容量。

### 1. 位（bit，缩写为 b）

- 定义：位是计算机中最小的信息单位，代表一个二进制数字（0 或 1），是 "二进制位"（binary digit）的缩写。
- 用途：用于表示数据的最基本状态，计算机底层的所有数据（文本、图像、声音等）最终都以位的形式存储和处理。

### 2. 字节（Byte，缩写为 B）

- 定义：字节是计算机中最常用的基本存储单位，1 字节 = 8 位（1B = 8b）。
- 用途：通常用于表示一个字符（如英文字母、数字或符号），例如在 ASCII 编码中，一个英文字母就占用 1 字节。

### 3. 字（Word）

- 定义：字是计算机进行数据处理时，一次能处理的二进制数据的长度，其大小与计算机的 CPU 架构有关，没有固定的标准。
- 常见情况：
- 32 位计算机：1 字 = 4 字节（32 位）
- 64 位计算机：1 字 = 8 字节（64 位）
- 用途：反映了 CPU 的处理能力，字长越大，CPU 一次能处理的数据量越大，运算效率通常越高。

### 4. KB（千字节）

- 定义：KB 是字节的倍数单位，1KB = 1024 字节（1KB = 1024B），这里采用二进制计数方式（2^10）。
- 注意：在部分场景（如硬盘厂商标注）中，可能会用 1KB = 1000B（十进制），但计算机领域通常默认 1024B。

### 5. 单位换算关系总结

- 1B（字节）= 8b（位）
- 1KB（千字节）= 1024B
- 1MB（兆字节）= 1024KB
- 1GB（吉字节）= 1024MB
- 1TB（太字节）= 1024GB
