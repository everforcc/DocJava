# OSI 七层模型（OSI/RM）核心考点总结

## 一、模型概述

- **定义**：由 ISO 提出的抽象七层模型，通过分层解耦解决不同设备的网络互连问题，是理解网络通信的基础框架。
- **核心思想**：每层仅与相邻层交互，屏蔽底层细节，提供标准化接口。
- **层序规则**：从下到上（物理层 → 应用层），数据传输时 “封装”（从上到下加头部），接收时 “解封装”（从下到上剥头部）。

## 三、各层关键细节补充

### 1. 物理层（最底层，硬件相关）

- 核心：不关心数据含义，仅传输 0/1 比特流，定义网线、接口、信号类型等物理规范。

- 易混点：集线器（Hub）是物理层设备，广播转发，所有端口共享带宽、同一冲突域。

### 2. 数据链路层（相邻设备通信）

- 核心机制：

- 帧封装：添加帧头（MAC 地址）+ 帧尾（CRC 校验）；

- 差错检测：CRC 仅检测不纠正错误；

- 介质访问控制：解决多设备共享介质的冲突问题。

- 易混点：交换机（二层设备）基于 MAC 地址表定向转发，每个端口独立冲突域，独享带宽。

### 3. 网络层（跨网段路由）

- 核心逻辑：

- IP 地址：逻辑地址，分网络位 + 主机位（子网掩码划分）；

- 路由表：路由器转发的依据（目标网段 → 下一跳）；

- ARP 协议：同一网段内将 IP 地址解析为 MAC 地址（网络层 → 数据链路层的桥梁）。

- 易混点：路由器（三层设备）连接不同网段，转发时更换 MAC 地址，不改变 IP 地址。

### 4. 传输层（承上启下）

- TCP 协议特性：

- 面向连接（三次握手建立，四次挥手释放）；

- 可靠传输（确认、重传、有序、流量控制 / 拥塞控制）；

- 适用场景：文件传输、网页、邮件（对可靠性要求高）。

- UDP 协议特性：

- 无连接、不可靠（无确认 / 重传）；

- 头部开销小、效率高；

- 适用场景：视频直播、语音通话、DNS 查询（对实时性要求高）。

- 五元组：源 IP + 源端口 + 协议 + 目的 IP + 目的端口（唯一标识一个端到端连接）。

### 5. 上三层（应用层 + 表示层 + 会话层）

- 共性：数据单位均为 “报文”，TCP/IP 模型中无单独分层，功能整合到应用层。

- 核心区别：

- 应用层：管 “业务服务”（如 HTTP 传网页）；

- 表示层：管 “数据格式”（如加密、编码）；

- 会话层：管 “对话控制”（如断点续传）。

## 四、高频易混点对比

### 1. MAC 地址 vs IP 地址

| 维度     | MAC 地址                    | IP 地址                |
| -------- | --------------------------- | ---------------------- |
| 类型     | 硬件地址（固化网卡）        | 逻辑地址（动态分配）   |
| 寻址范围 | 同一局域网（相邻设备）      | 跨网段（全局定位主机） |
| 管理方   | IEEE 分配（OUI + 扩展标识） | 运营商 / 管理员分配    |

### 2. 物理层设备 vs 数据链路层设备 vs 网络层设备

| 设备             | 工作层次   | 核心能力                 | 冲突域特性                  |
| ---------------- | ---------- | ------------------------ | --------------------------- |
| 集线器（Hub）    | 物理层     | 信号放大 + 广播转发      | 所有端口同一冲突域          |
| 交换机（Switch） | 数据链路层 | MAC 地址定向转发         | 每个端口独立冲突域          |
| 路由器（Router） | 网络层     | IP 路由选择 + 跨网段转发 | 每个端口独立冲突域 + 广播域 |

### 3. OSI 模型 vs TCP/IP 模型

| OSI 七层模型 | TCP/IP 四层模型      | 对应关系                         |
| ------------ | -------------------- | -------------------------------- |
| 应用层       | 应用层               | TCP/IP 应用层整合 OSI 上三层功能 |
| 表示层       | 应用层               |                                  |
| 会话层       | 应用层               |                                  |
| 传输层       | 传输层               | 功能一致（TCP/UDP）              |
| 网络层       | 网络层（网际层）     | 功能一致（IP / 路由）            |
| 数据链路层   | 网络接口层（链路层） | 功能一致（帧 / MAC）             |
| 物理层       | 网络接口层（物理层） | 功能一致（比特流）               |
