│  📌 典型设备：无独立设备（功能嵌入应用/中间件）                     │

│  📌 典型设备：无独立设备（功能由操作系统内核实现）                   │
│  📋 典型协议：TCP、UDP                                             │

│  📋 典型协议：IP(v4/v6)、ICMP、ARP、RARP、OSPF、RIP                 │
│  📦 数据单位：数据包（Packet）                                      │

│  📋 典型协议：Ethernet、PPP、HDLC、802.11(WiFi)                     │
│  📦 数据单位：帧（Frame）                                           │

│  📌 典型设备：集线器（Hub）、中继器、网线、光纤、网卡物理接口        │
│  📋 典型协议/标准：RJ45、RS-232、IEEE 802.3                         │
│  📦 数据单位：比特（Bit）                                           │
