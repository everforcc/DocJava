### TCP/IP 分层与核心协议

- 网络层：IP、ARP、RARP、ICMP、IGMP。
- 传输层：TCP、UDP。
- 应用层：HTTP/HTTPS、FTP、SMTP/POP3/IMAP、DNS、DHCP、SNMP、TELNET/SSH 等。

### 端口与常见默认值

- HTTP 80、HTTPS 443、FTP 21/20、SSH 22、SMTP 25、DNS 53、DHCP 67/68、SNMP 161/162。

### TCP 与 UDP 对比

- TCP：面向连接、可靠、有序、流量控制、拥塞控制，适合文件、网页。
- UDP：无连接、尽力而为、无序、无拥塞控制，适合实时音视频、DNS 查询。
