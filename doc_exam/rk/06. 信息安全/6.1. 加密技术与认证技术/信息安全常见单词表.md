### 信息安全高频易混淆缩写对照表（Markdown 版）

| 缩写      | 英文全拼                               | 中文译名             | 核心定义                                                           | 应用场景                               | 软考高频考点                          |
| :-------- | :------------------------------------- | :------------------- | :----------------------------------------------------------------- | :------------------------------------- | :------------------------------------ |
| **MAC**   | Message Authentication Code            | 消息认证码           | 基于密钥的消息认证技术，生成固定长度数据用于验证消息完整性和真实性 | 内部系统通信、API 接口认证、IPsec 协议 | 与数字签名的区别、HMAC 原理           |
| **MAC**   | Media Access Control                   | 介质访问控制         | 数据链路层的子层，负责控制网络节点对传输介质的访问                 | 以太网、Wi-Fi 等局域网通信             | 以太网 MAC 地址格式、CSMA/CD 协议     |
| **HMAC**  | Hash-based Message Authentication Code | 基于哈希的消息认证码 | 结合哈希函数和对称密钥的消息认证机制                               | HTTP 接口签名、SSL/TLS 协议、金融交易  | 工作原理、安全性分析                  |
| **CA**    | Certificate Authority                  | 证书颁发机构         | 负责颁发和管理数字证书的权威机构，验证公钥持有者身份               | PKI 体系、HTTPS 网站、数字签名         | 数字证书生命周期、CA 信任模型         |
| **PKI**   | Public Key Infrastructure              | 公钥基础设施         | 基于公钥密码技术的安全体系，包含 CA、证书库、密钥管理等组件        | 身份认证、数据加密、数字签名           | 组成结构、核心功能                    |
| **PMI**   | Privilege Management Infrastructure    | 权限管理基础设施     | 管理用户访问权限的安全体系，与 PKI 配合实现细粒度访问控制          | 企业权限管理、系统安全访问             | 与 PKI 的区别与协同                   |
| **SSL**   | Secure Sockets Layer                   | 安全套接层           | 用于在网络通信中提供加密和认证的协议                               | 早期 HTTPS、邮件加密                   | 与 TLS 的关系、握手过程               |
| **TLS**   | Transport Layer Security               | 传输层安全           | SSL 的升级版，提供更安全的传输层加密和认证                         | HTTPS、VPN、即时通信                   | 协议分层、加密套件选择                |
| **IPsec** | Internet Protocol Security             | 互联网协议安全       | 用于 IP 层安全通信的协议簇，提供加密、认证和完整性保护             | 虚拟专用网（VPN）、企业内网通信        | 协议组成（AH/ESP）、工作模式          |
| **IDS**   | Intrusion Detection System             | 入侵检测系统         | 监控网络或系统，识别并报告可疑活动或攻击行为                       | 网络安全监控、异常行为检测             | 分类（基于签名/异常）、与 IPS 的区别  |
| **IPS**   | Intrusion Prevention System            | 入侵防御系统         | 基于 IDS 的主动防御系统，可实时阻断可疑流量                        | 网络边界防护、恶意攻击拦截             | 与 IDS 的核心差异、防御机制           |
| **VPN**   | Virtual Private Network                | 虚拟专用网络         | 通过公共网络建立加密的专用通信通道                                 | 远程办公、跨组织安全通信               | 实现技术（IPsec/SSL）、核心功能       |
| **ACL**   | Access Control List                    | 访问控制列表         | 用于控制资源访问权限的规则集合                                     | 路由器防火墙、操作系统权限管理         | 工作原理、规则匹配顺序                |
| **RBAC**  | Role-Based Access Control              | 基于角色的访问控制   | 按用户角色分配访问权限的模型，简化权限管理                         | 企业应用、大型系统权限控制             | 模型核心要素（用户/角色/权限）        |
| **DES**   | Data Encryption Standard               | 数据加密标准         | 经典的对称分组加密算法，密钥长度 56 位                             | 早期数据加密、历史加密场景             | 加密原理、安全性缺陷（已被 AES 取代） |
| **AES**   | Advanced Encryption Standard           | 高级加密标准         | 目前主流的对称分组加密算法，支持 128/192/256 位密钥                | 数据加密、SSL/TLS、存储加密            | 轮函数结构、密钥长度选择              |
| **RSA**   | Rivest-Shamir-Adleman                  | 非对称加密算法       | 基于大整数分解难题的公钥加密算法                                   | 数字签名、密钥交换、证书加密           | 加密/解密过程、安全性基础             |
| **DSA**   | Digital Signature Algorithm            | 数字签名算法         | 基于离散对数难题的数字签名算法                                     | 数字签名、身份认证                     | 与 RSA 签名的区别、应用场景           |
