### Windows 常用

- ipconfig /all：查看地址、网关、DNS。
- ping：连通性与时延；`-n` 次数、`-l` 大小。
- tracert：路径探测（逐跳 TTL）。
- netstat -ano：端口与 PID；ESTABLISHED/TIME_WAIT 等状态。
- nslookup：DNS 解析；可指定服务器。

### Linux 常用

- ifconfig/ip addr、route/ip route、tcpdump、ss、mtr。

### 典型排障流程

1) 本机地址与网关是否正确。
2) DNS 能否解析域名。
3) ping 网关/外网；traceroute 定位丢包节点。
4) 抓包比对握手与应用层交互。
