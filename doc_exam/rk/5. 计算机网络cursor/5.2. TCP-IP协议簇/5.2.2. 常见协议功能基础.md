### IP/ICMP/ARP

- IP：无连接、尽力转发；子网划分、路由选择。
- ICMP：差错与控制（目的不可达、超时报文、回显请求/应答用于 ping）。
- ARP：解析同一链路内 IP→MAC。

### DNS/DHCP

- DNS：域名解析，UDP/53（查询），也可 TCP。
- DHCP：自动分配地址、网关、DNS；UDP 67/68；四步：DISCOVER→OFFER→REQUEST→ACK。

### HTTP/HTTPS

- HTTP 明文无状态；HTTPS = HTTP + TLS，提供机密性与完整性与身份认证。

### SMTP/POP3/IMAP

- SMTP 负责“发送/转发邮件”；POP3/IMAP 负责“客户端收取/同步”。
