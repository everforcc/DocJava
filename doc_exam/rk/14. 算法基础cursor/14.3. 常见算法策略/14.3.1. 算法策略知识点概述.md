# 14.3.1. 算法策略知识点概述

## 概述
算法策略是解决计算问题的基本方法和思路，是算法设计的核心思想。掌握不同的算法策略，能够帮助我们选择合适的方法来解决各种复杂问题，提高算法设计的效率和质量。

## 算法策略的分类

### 1. 分治法（Divide and Conquer）
将问题分解为若干个相同或相似的子问题，递归地解决子问题，然后将子问题的解合并得到原问题的解。

### 2. 动态规划法（Dynamic Programming）
通过将问题分解为相互重叠的子问题，并存储子问题的解，避免重复计算，从而高效地解决问题。

### 3. 贪心法（Greedy Algorithm）
在每一步都做出在当前看来最好的选择，希望通过局部最优解达到全局最优解。

### 4. 回溯法（Backtracking）
通过系统地搜索所有可能的解，当发现当前路径不能得到解时，回退到上一步，尝试其他路径。

### 5. 分支限界法（Branch and Bound）
在回溯法的基础上，通过剪枝来减少搜索空间，提高搜索效率。

## 分治法

### 基本思想
1. **分解**：将原问题分解为若干个规模较小的子问题
2. **解决**：递归地解决各个子问题
3. **合并**：将子问题的解合并为原问题的解

### 适用条件
- 问题可以分解为相同类型的子问题
- 子问题相互独立
- 子问题的解可以合并为原问题的解

### 典型应用
- 归并排序
- 快速排序
- 二分查找
- 大整数乘法
- 矩阵乘法

### 时间复杂度分析
通常使用递归关系式：T(n) = aT(n/b) + f(n)
- a：子问题个数
- n/b：子问题规模
- f(n)：分解和合并的时间

## 动态规划法

### 基本思想
1. **最优子结构**：问题的最优解包含子问题的最优解
2. **重叠子问题**：递归过程中会重复计算相同的子问题
3. **状态转移**：通过状态转移方程描述问题的最优解

### 基本步骤
1. 定义状态
2. 建立状态转移方程
3. 确定初始状态
4. 计算最优解

### 典型应用
- 最长公共子序列
- 最长递增子序列
- 背包问题
- 最短路径问题
- 编辑距离

### 与分治法的区别
- 分治法：子问题相互独立
- 动态规划：子问题相互重叠

## 贪心法

### 基本思想
在每一步都选择当前看起来最优的选择，不考虑后续步骤的影响。

### 适用条件
- 问题具有最优子结构
- 贪心选择性质：局部最优解能导致全局最优解

### 基本步骤
1. 建立数学模型
2. 证明贪心选择性质
3. 证明最优子结构
4. 设计贪心算法

### 典型应用
- 活动选择问题
- 最小生成树
- 最短路径（Dijkstra算法）
- 哈夫曼编码
- 区间调度问题

### 局限性
- 不能保证得到全局最优解
- 需要证明贪心选择性质

## 回溯法

### 基本思想
通过系统地搜索所有可能的解空间，当发现当前路径不能得到解时，回退到上一步，尝试其他路径。

### 基本步骤
1. 定义解空间
2. 确定搜索策略
3. 设计约束条件
4. 实现回溯过程

### 典型应用
- N皇后问题
- 数独求解
- 图的着色问题
- 旅行商问题
- 子集和问题

### 优化策略
- 剪枝：提前终止不可能得到解的分支
- 约束传播：利用约束条件减少搜索空间
- 启发式搜索：使用启发式信息指导搜索

## 分支限界法

### 基本思想
在回溯法的基础上，通过计算每个节点的界限值，剪枝那些不可能得到最优解的分支。

### 基本步骤
1. 定义界限函数
2. 选择扩展节点
3. 计算界限值
4. 剪枝和回溯

### 典型应用
- 0-1背包问题
- 旅行商问题
- 作业调度问题
- 图着色问题

### 与回溯法的区别
- 回溯法：深度优先搜索
- 分支限界法：广度优先搜索或最佳优先搜索

## 算法策略的选择

### 选择原则
1. **问题特征**：根据问题的特点选择合适策略
2. **时间复杂度**：考虑算法的时间复杂度
3. **空间复杂度**：考虑算法的空间复杂度
4. **实现难度**：考虑算法的实现复杂度

### 选择指南
- **分治法**：问题可以分解为独立子问题
- **动态规划**：问题具有重叠子问题和最优子结构
- **贪心法**：问题具有贪心选择性质
- **回溯法**：需要搜索所有可能解
- **分支限界法**：需要找到最优解且可以剪枝

## 算法策略的比较

### 时间复杂度
| 策略 | 典型时间复杂度 | 适用场景 |
|------|----------------|----------|
| 分治法 | O(n log n) | 可分解问题 |
| 动态规划 | O(n²) 或 O(n³) | 优化问题 |
| 贪心法 | O(n log n) | 选择问题 |
| 回溯法 | O(2ⁿ) | 搜索问题 |
| 分支限界法 | O(b^d) | 优化搜索 |

### 空间复杂度
| 策略 | 典型空间复杂度 | 特点 |
|------|----------------|------|
| 分治法 | O(log n) | 递归栈空间 |
| 动态规划 | O(n) 或 O(n²) | 存储子问题解 |
| 贪心法 | O(1) | 常数空间 |
| 回溯法 | O(n) | 递归栈空间 |
| 分支限界法 | O(b^d) | 队列空间 |
