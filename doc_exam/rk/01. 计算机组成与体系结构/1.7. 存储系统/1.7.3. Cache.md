### 直接相联映像下

- todo映射图
- 主存地址的结构
- 主存地址需要被拆分为 3 个部分，用于定位数据并判断 Cache 是否命中：
- 主存地址 = 标记（Tag） + Cache行号（Line Number） + 块内偏移（Offset）
- **块内偏移（Offset）**
    - 长度由 “块大小” 决定。例如，若块大小为 64B（2⁶B），则块内偏移需要 6 位（可表示 0~63，对应块内 64 个字节）。作用是定位
      “块内的具体字节”。
- **Cache 行号（Line Number）**
    - 长度由 “Cache 总行数” 决定。例如，若 Cache 有 8 行（2³ 行），则行号需要 3 位（可表示 0~7，对应 8 个行）。作用是确定
      “当前主存块应该映射到 Cache 的哪一行”。
- **标记（Tag）**
    - 主存地址中剩下的高位部分。作用是 “区分映射到同一 Cache 行的不同主存块”（因为多个主存块可能映射到同一个 Cache
      行，需要用标记判断是否为当前访问的块）。
