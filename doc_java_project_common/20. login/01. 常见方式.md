### http无状态协议

- 为了处理有状态的需求，引入了一些机制，如Cookie和会话（Session）。
- Cookie是一种在客户端存储数据的机制，而会话是一种在服务器端存储数据的机制。
- 通过将会话标识符存储在Cookie中或通过URL重写等方式将其传递，服务器可以识别客户端并恢复相关的状态信息
