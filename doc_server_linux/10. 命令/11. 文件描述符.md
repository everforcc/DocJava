### 文件描述符

~~~shell
nohup java -jar app.jar >log 2>&1 &
# 简写
nohup java -jar app.jar &>log &
~~~

#### > log

- > log和>>log区别

| 操作符    | 行为描述                   | 是否覆盖原文件 | 示例                     |
|---------|----------------------------|----------------|--------------------------|
| `> log`   | 输出重定向（覆盖写入）     | ✅ 是           | `java -jar app.jar > log`   |
| `>> log`  | 输出重定向（追加写入）     | ❌ 否           | `java -jar app.jar >> log`  |

#### 2>&1

- 2>&1 是 Linux/Unix 系统中用于处理 文件描述符 的语法，它的含义是：
    - 2：标准错误输出（stderr）的文件描述符。
    - \> ：重定向操作符。
    - &1：引用标准输出（stdout）当前的值。
- 将原本输出到终端的标准错误信息（stderr），重定向到和标准输出（stdout）相同的位置。

| 2          | stderr           | 标准错误输出（Standard Error Output）| 终端显示器         |
