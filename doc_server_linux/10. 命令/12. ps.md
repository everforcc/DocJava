### ps

~~~
ps aux | grep "docker logs --tail 100 -f 2c7052a94736"
ps aux | grep "docker logs"
user123 12345 0.0 0.1 123456 7890 pts/0 S+ 10:00 0:00 docker logs --tail 100 -f 2c7052a94736

kill 12345
kill -9 12345
~~~

| 命令                      | 说明                 |
|-------------------------|--------------------|
| `ps aux`                | 查看所有进程详细信息         |
| `ps -ef`                | 查看所有进程（UNIX 风格）    |
| `ps -p PID`             | 查看指定 PID 的进程信息     |
| `ps aux \| grep java`   | 查找包含 `"java"` 的进程  |
| `ps -ef \| grep docker` | 查找与 `docker` 相关的进程 |
| `ps aux \| grep "关键字"`  | 查找包含特定关键词的进程       |
| `ps -aux --sort=-%mem`  | 按内存使用降序排列进程        |
| `ps -aux --sort=-%cpu`  | 按 CPU 使用降序排列进程     |
