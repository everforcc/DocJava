### 网关 nginx

- 前后端分离模式部署
- 前端代码发布在nginx
- 通过反向代理访问后端
- 通过权重配置多个网关服务

### 缓存 redis

- 使用redis集群缓存中间件，对经常访问的数据进行缓存，例如缓存用户token，降低数据库压力
- 使用Redis-Cluster模式，不同数据分布在不同的服务器，三主三从
- 各个服务配置所有节点

### 数据库 mysql

- 使用 主-从1-从2 模式，主库处理写入和读取数据，从1只负责读取数据
- 从2负责每日备份数据
