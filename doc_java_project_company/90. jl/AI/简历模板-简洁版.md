## 使用说明

### 填写要点
1. **基本信息**：填写真实信息，联系方式要准确
2. **专业技能**：根据实际掌握情况调整，突出核心技能
3. **工作经历**：按时间倒序排列，重点描述职责和成果
4. **项目经验**：选择最能体现技术能力的项目，详细描述
5. **教育背景**：如实填写，如有相关证书可在此处展示

### 优化建议
- 用具体数字替代模糊描述
- 重点描述解决了什么问题，带来了什么价值
- 突出技术难点和解决方案
- 确保所有内容真实可信
- 根据目标公司要求调整关键词
