### 大文件分片上传

- [大文件秒传、断点续传和分片上传](https://mp.weixin.qq.com/s/4ty-x7MYKU_saPaavikN5w)

### 上传流程

1. 前端生成文件的MD5值，分片计算
2. 判断当前的文件上传信息
3. 每个分片生成临时的凭证
4. 前端上传分片和请求后端合并文件
5. 秒传是因为之前这个文件已经上传过，数据库中已经存储了文件在MinIO上完整的地址。
6. 断点续传的原理是记录已经完成上传的分片，再次上传的时候这些分片无需再次上传，只上传未完成上传的分片。
