### 5. 编写定时任务需要执行的备份脚本

- 一开始是想通过 docker内 dump脚本备份然后映射到宿主机，但是不会docker内安装 cron定时任务
- 随后通过 docker /bin/bash -c 的方式，直接将备份文件写到宿主机内

~~~
# 保存到指定目录内 
/03. backup_script.sh
# 需要开启权限
chmod +x backup.sh
~~~

#### 5.1 压缩

- 可选
- 对备份后的脚本进行压缩

### 6. 设置定时任务

~~~
crontab -e
# 每分钟
*/1 * * * * /pathto/mysql/test/backup.sh  >> /var/log/cron.log 2>&1
# 每天 凌晨两点执行定时任务
0 2 * * * /pathto/mysql/shell/backup.sh  >> /var/log/cron.log 2>&1
~~~
