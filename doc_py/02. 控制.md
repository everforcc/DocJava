### if

~~~
if condition_1:
    statement_block_1
elif condition_2:
    statement_block_2
else:
    statement_block_3
~~~

### match...case

~~~
match subject:
    case <pattern_1>:
        <action_1>
    case <pattern_2>:
        <action_2>
    case <pattern_3>:
        <action_3>
    case _:
        <action_wildcard>
~~~

### while

~~~
while <expr>:
    <statement(s)>
else:
    <additional_statement(s)>
~~~

- 没有 do while

### for

~~~
#!/usr/bin/python3
 
sites = ["Baidu", "Google","Runoob","Taobao"]
for site in sites:
    if site == "Runoob":
        print("菜鸟教程!")
        break
    print("循环数据 " + site)
else:
    print("没有循环数据!")
print("完成循环!")
~~~

### range() 函数

~~~
# 如果你需要遍历数字序列，可以使用内置 range() 函数。它会生成数列
for i in range(5):
     print(i)
~~~

- 你也可以使用 range() 指定区间的值： range(5,9)
- 也可以使 range() 以指定数字开始并指定不同的增量(甚至可以是负数，有时这也叫做'步长'):
    - range(0, 10, 3)
