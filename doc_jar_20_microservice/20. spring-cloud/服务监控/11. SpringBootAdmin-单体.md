- 配置文件

~~~yaml
# 暴露监控端点
# 单机
# 还需要增加注册地址
logging:
  file:
    name: logs/cc-gateway/info.log
# 微服务通过nacos注册
management:
  endpoints:
    web:
      exposure:
        include: '*'
  endpoint:
    health:
      show-details: always
    shutdown:
      enabled: true
    # 微服务
    logfile:
      external-file: logs/cc-gateway/info.log
~~~
