### 几种情况

- 非会员
    - 无用户录入
        - 支付宝
        - 现金
    - 用户录入
        - 支付宝
        - 现金
- 会员
- 卡券*用户录入
    - 无卡券 * 无用户录入
    - 无卡券 * 有用户录入
    - [x] 有卡券 * 无用户录入
    - [x] 有卡券 * 有用户录入
- 支付宝/现金
    - 都记作充值记录
