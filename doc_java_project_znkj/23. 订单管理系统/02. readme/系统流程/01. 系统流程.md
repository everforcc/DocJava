### 市场管理

#### 市场经理

- 市场经理由中南直接创建
- 可以修改
- 可以删除后替换为另一个人

#### 部门业绩

- 市场经理下的城市合伙人的充值
- 归为市场经理的业绩

##### 经销商加入标准

- 可以设置不同地区的城市合伙人和地区代理商的保证金金额

### 经销商管理

#### 加盟申请审核

- 扫市场经理APP的招商码，成为城市合伙人
- 扫城市合伙人APP的招商码，成为地区代理商
- 审核通过拒绝都会发短信
- 每个地区都只能由一个城市合伙人和地区代理商

#### 经销商管理

- 可以根据市场经理和城市合伙人
- 搜索自己名下的城市合伙人和地区代理商

#### 投诉建议

- 可以查看app提交的投诉建议

### 财务管理

#### 分成与奖励

##### 经销商首冲奖励

- 不同额度首充金额赠送金豆设置

##### 中南分成设置

- 中南和城市合伙人分成
- 售卖类的金豆分成
- 租赁类的现金分成

##### 中南市场经理提成

- 可以设置按季度的晋升额度
- 也可以设置提成比例

#### 充值与保证金

- 保证金，充值金额，提货金额，剩余金额
- 全部用户数据汇总
- 每个用户数据明细

#### 收款账户

- 中南收款信息配置
