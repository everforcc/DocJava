### 产品管理

#### 类别系列型号管理

- 分为出售和租赁两类
- 类别
- 类别下面可以新增系列

#### 产品配置库

##### 产品配置管理

- 可以配置产品功能
- 产品配套
- 配套赠送
- 等属性

##### 新增型号

- 型号属性从产品配置管理里面选择
- 出售和租赁都可以设置型号

#### 产品管理

- 每个产品型号只能新增一个产品
- 编辑好的商品需要审核
- 审核通过的商品可以上架
- 已上架的商品只有下架后才能重新编辑
- 已审核已上架的商品可以在app上面购买

#### APP首页banner

- 最多可以有三个生效
- 设置banner图和详情内容
