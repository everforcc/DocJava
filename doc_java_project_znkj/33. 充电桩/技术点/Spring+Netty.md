### 1. JVM进程层面

- Spring Boot应用运行在一个JVM进程中
- XiaoyuServer 启动的Netty服务运行在同一个JVM进程内
- 进程级别的 ApplicationContext 对整个应用可见

### 2. 线程层面

- Netty的 EventLoopGroup 创建多个I/O线程
- 这些线程都在同一个进程的内存空间内运行
- 线程可以访问进程内的Spring容器

### 3. 内存共享机制

~~~text
┌─────────────────────────────────────┐
│        JVM进程 (Spring Boot应用)     │
│                                     │
│  ┌─────────────────────────────┐    │
│  │   Spring容器                │    │
│  │  ┌─────────────────────┐    │    │
│  │  │    Bean实例         │    │    │
│  │  └─────────────────────┘    │    │
│  └─────────────────────────────┘    │
│                                     │
│  ┌─────────────────────────────┐    │
│  │   Netty线程池               │    │
│  │   ┌─────────────────────┐   │    │
│  │   │   I/O线程1         │   │    │
│  │   │   I/O线程2         │   │    │
│  │   │   I/O线程3         │   │    │
│  │   └─────────────────────┘   │    │
│  └─────────────────────────────┘    │
└─────────────────────────────────────┘
~~~

### 4. 关键理解

- 进程：提供共享的内存空间和Bean容器
- 线程：执行具体的I/O操作，可访问进程内的所有资源
- Netty线程虽然处理网络I/O，但仍在Spring容器的进程范围内

### 总结

- 所以这里的"进程"指的是整个Spring Boot应用的JVM进程，而不是Netty单独的进程。
