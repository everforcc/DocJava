### 小时

- 用户选择几个小时
- 从余额扣除
- 生成订单，初始化

### 下发充电

#### 异常

- 订单异常
- 退回余额
- 结束

#### 开始充电

##### 时间/度 用完

- 设备上报断电
- 钱算出电费/服务费

##### 充满自停

- 设备上报充满

##### 远程断电

- 用户远程断电

#### 计费逻辑

- 预扣费
- 用完
    - 订单结束
- 用不完
    - 充满自停
    - 远程断电
- 时
    - [ ]  按比例算出电费/服务费
    - [ ]  退回余额
- 度
    - [ ]  按比例算出电费/服务费
    - [ ]  退回余额
