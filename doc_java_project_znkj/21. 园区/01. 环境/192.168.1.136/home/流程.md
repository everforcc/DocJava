### 1. 编写清理脚本

~~~
# 创建脚本文件
sudo nano /home/clean_ruoyi.sh
sudo nano clean_ruoyi.sh
sudo nano list_ruoyi.sh


#!/bin/bash
# 删除以 ruoyi20 开头的目录
find /home/smartparkcloud-docker -type d -name "ruoyi20*" -exec rm -rf {} + 2>/dev/null

~~~

### 2. 给脚本添加执行权限

~~~
# 777
chmod 755 /home/clean_ruoyi.sh
~~~

### 3. 配置 crontab 定时任务

~~~
crontab -e
0 2 * * * /home/clean_ruoyi.sh
~~~

### 4.  验证 crontab 配置

~~~
crontab -l

~~~
