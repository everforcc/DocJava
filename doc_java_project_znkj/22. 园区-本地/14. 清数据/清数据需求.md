### 使用技术

- mybaties
- @SelectProvider
- @DeleteProvider

### 定义

- 给定一个表名转，可以动态拼接，例如 zn_table，公共字段 create_time date 类型， id bigint 主键
- 给定一个int类型的参数，例如 max_count
- 给定两个double类型的参数，max_rate,min_rate

### 业务逻辑

- @SelectProvider动态拼接表名查询
- 查询当前表中数据量 now_count，如果小于 max_count * max_rate不作处理，
- 如果大于 max_count * max_rate，则删除数据，
- 删除规则，按照create_time升序，删除 now_count-max_count * min_rate的数据量
- 查询出上一步数据量的id
- @DeleteProvider动态拼接表名，批量删除
