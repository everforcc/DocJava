### 顶层字段说明：

| 关键词 |     描述     | 类型 | 说明                       |
| :----: | :----------: | :---: | -------------------------- |
|  “gw”  | 网关消息标识 | 对象 | 网关JSON数据包含在该字段下 |

### gw下字段说明：

|  关键词  |     描述     |  类型  | 说明                                                         |
| :------: | :----------: | :----: | ------------------------------------------------------------ |
|  “gwid”  |   网关地址   | 字符串 | 8字节十六进制字符串，即16个字符                              |
|  “gwpv”  | 网关协议版本 | 字符串 | 版本字符串                                                   |
|  “gwdv”  | 网关设备版本 | 字符串 | 版本字符串                                                   |
|  “gwsq”  |  网关包序号  |  数字  | 4字节无符号整型 其取值在节点数据主动上报流程（交互流程说明见第3章交互流程）中，该序号由网关生成，操作节点、操作网关流程中取下行数据的gwsq字段 |
| “gwifty” | 网关消息类型 |  数字  | 0 射频包数据 1 网关配置回复 2 4G终端 99 网关心跳报文 100 网关注册报文 |
|  “gwak”  | 网关配置回复 |  数组  | 只在gwifty为1时有效                                          |
|  “rxpk”  |  射频包数据  |  数组  | 只在gwifty为0或10时有效                                      |

### gwak下字段说明：

|  关键词  |   描述   |  类型  | 说明                     |
| :------: | :------: | :----: | ------------------------ |
| “gwakob” | 执行对象 | 字符串 | 值为定义的执行对象字符串 |
| “gwakrt” | 执行结果 |  数字  | 0执行成功，1执行失败     |
| “gwakif” | 附带信息 | 字符串 | 预留                     |

### rxpk下字段说明：

| 关键词 |    描述    |  类型  | 说明                                                         |
| :----: | :--------: | :----: | ------------------------------------------------------------ |
| “tmst” |   时间戳   | 字符串 | 射频计时器值                                                 |
| “time” |  接收时间  | 字符串 | 合法的时间戳字符串，单位为秒                                 |
| “chan” |   信道号   |  数字  | 无符号整型                                                   |
| “freq” |    频点    |  数字  | 无符号整型数字代表频点，单位Hz                               |
| “foff” |    频偏    |  数字  | 1.1.0协议版本支持，有符号整形数字，单位Hz                    |
| “rssi” |  信号强度  |  数字  | 有符号整型，单位dBm                                          |
| “lsnr” |   信噪比   |  数字  | 浮点型                                                       |
| “rfch” |   通道号   |  数字  | 无符号整型数字                                               |
| “stat” |    状态    |  数字  | 无符号整型数字, 无线CRC状态                                  |
| “modu” |  调制模式  | 字符串 | LORA”、”FSK”                                                 |
| “datr” |  扩频因子  | 字符串 | 取值范围为”SF5”、”SF6”、”SF7”、”SF8”、”SF9”、”SF10”、”SF11”、”SF12” |
| “codr” |   编码率   | 字符串 | 取值范围为”4/5”、”4/6”、”4/7”、”4/8”                         |
| “pksz” | 数据包长度 |  数字  | 无符号整型数字，不超过256                                    |
| “pkdt” | 数据包内容 | 字符串 | 十六进制字符串，长度为pksz*2                                 |

### pkdt

- 36位数,18组
- 0
    - 协议头
- 1234
    - 设备号
- 5
    - 控制字
- 67
    - 包序号
- 89
    - 设备信息和10结合使用
- 10
    - 传感器类型
- 11 12 13
    - 温度
- 14 15 湿度
- 16 17
- CRC校验
