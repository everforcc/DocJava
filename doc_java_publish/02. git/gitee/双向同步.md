### 双向同步

- [gitee双向同步](https://gitee.com/help/articles/4336)
- 操作步骤

#### 手动

- 项目从github拉取最新代码
- 项目名右边的同步按钮

#### 自动

- 拉取
- 推送

1. 打开 Gitee 仓库设置页面
2. 点击管理，找到 仓库镜像管理 目录
3. 添加镜像
4. 选择镜像仓库
5. 添加 github 令牌

### github权限

- 您的私人令牌中需要包含对 repo 和 admin:repo_hook（ 用于自动生成 webhook ）的访问授权
- repo
- admin:repo_hook（ 用于自动生成 webhook ）
